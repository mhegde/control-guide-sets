Finding essential control guides for CRISPRi using exisiting Dolcetto data¶

The Dolcetto Set A data from Sanson et al., 2019 was used in this analysis. The average log-fold change of 3 replicates for HT29 and A375 was calculated. The essential genes list from Hart et al., 2015 and non-essential genes list from Hart et al., 2014 was used to determine guides targeting essential and non-essential genes.
